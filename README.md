# Asaan
This repository contains the code for Asaan's website.

## Requirements

1. Node.js
2. git
3. npm

## Step 1
clone this repository:

```bash
git clone https://github.com/asaanhq/website.git
```

## Step 2
Install all dependencies:

```bash
cd website
npm i
```

## Step 3
To run in development mode:

```bash
npm start
```
